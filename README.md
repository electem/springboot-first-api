Project Name: springboot-first-api

Description:
This is a basic Spring Boot project created for learning purpose.
In this project we created a simple API using Controller and Service layer.
This project helps beginners to understand how to create and run a Spring Boot application and how API works.

Technologies Used:
Java
Spring Boot
Spring Web
Lombok (optional)

Project Structure:

controller package
Used to handle API requests and send response

service package
Used to write business logic

repository package
Used for database operations (not used in this project)

API Details:

API Name: Hello API
Method: GET
URL: http://localhost:8080/api/hello

Response:
Hello World

How to Run the Project:

Download or clone the project
Open project in IDE like IntelliJ or Eclipse
Make sure Java version is properly installed
Wait until dependencies are downloaded
Run main class (DemoApplication.java)
Application will start on port 8080
Open browser or Postman and call the API

Expected Output:

When you call the API, you will get response as
Hello World

Notes:

Spring Web dependency is required to create APIs
Controller is used to handle requests
Service is used for business logic
Repository is not used in this example

This is a beginner level project to understand basic API creation in Spring Boot
